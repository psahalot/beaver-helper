A nifty plugin to add useful features and admin bar links for Beaver Builder plugin.

== Description ==

Beaver Helper plugin adds quick links to admin bar for easy access to
* BB Facebook Group
* BB Slack Channel
* Knowledge Base
* Official Support

I am working on adding more handful of features like:
* Quick Preview
* Keyboard Shortcuts
* Module Search

Check out useful Beaver Builder addons at [WP Beaver Addons](https://wpbeaveraddons.com) to speed up your Web Development process.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/bb-helper` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= What's this plugin for? =

This plugin is an add-on for Beaver Builder plugin to enhance it's functionality.

= Will it work with both Free & Paid versions of Beaver Builder? =

Yes, it works with both free and paid versions of Beaver Builder.

= I don't see any links in the admin bar, what's wrong? =

Please make sure that either Beaver Builder Lite Version or the paid version plugin is active.
